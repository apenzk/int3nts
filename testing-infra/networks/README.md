# Network Deployment

Deploy and configure scripts for testnet and mainnet environments.

- **[testnet/](./testnet/)** — Testnet deployment (Movement Bardock, Base Sepolia, Solana Devnet)
- **mainnet/** — Mainnet deployment (Movement Mainnet, Base Mainnet, HyperEVM Mainnet)
- **common/** — Shared scripts and utilities sourced by both testnet and mainnet wrappers

See [supported-networks.md](../../docs/testing-infra/supported-networks.md) for supported networks and deployment cost estimates.
